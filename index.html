제안 개요

제안 목적

여성가족부 공공데이터를 활용하여 청소년의 미디어 이용 행태와 유해환경 노출 위험성을 분석하고, 이를 기반으로 개인별 맞춤형 디지털 미디어 리터러시 교육을 제공하는 AI 기반 플랫폼을 개발합니다.

이를 통해 청소년의 건강한 미디어 이용 습관을 형성하고, 사이버 폭력, 디지털 과의존, 유해정보 노출 등의 부정적 영향을 최소화하며, 디지털 시민으로서의 역량을 강화하고자 합니다.

비판적 사고와 창의적 미디어 활용 능력 향상

데이터 활용

청소년종합실태조사

공공데이터포털에서 제공하는 여성가족부 청소년종합실태조사 마이크로데이터를 활용합니다.

API 엔드포인트: http://apis.data.go.kr/1383000/stis/srvyTeenOvrlMcrDataService

주요 활용 항목

  • 청소년의 일상생활
  • 가정환경
  • 학교생활
  • 여가활동
  • 심리상태

청소년매체이용및유해환경실태조사

AI 기술 활용 방안

사용자 유형 분류 모델

기술: 클러스터링 알고리즘(K-means)

입력 데이터: 미디어 이용 시간, 플랫폼, 목적, 콘텐츠 유형 등

출력 결과: 사용자 그룹 분류(예: 정보 탐색형, 소셜 활동형, 엔터테인먼트 중심형)

활용: 그룹별 특화 교육 콘텐츠 설계

위험요소 예측 모델

기술: 분류 알고리즘(랜덤 포레스트, XGBoost)

입력 데이터: 미디어 이용 패턴, 인구통계학적 특성, 유해환경 노출 이력

출력 결과: 위험 유형별 점수(디지털 과의존, 유해콘텐츠 노출, 사이버 폭력 등)

활용: 맞춤형 예방 교육 및 대응 전략 제시

콘텐츠 추천 시스템

기술: 하이브리드 추천 알고리즘(콘텐츠 기반, 협업 필터링)

입력 데이터: 사용자 프로필, 학습 이력, 피드백, 유사 사용자 행동

출력 결과: 개인별 최적화된 교육 콘텐츠 및 학습 경로

활용: 효과적이고 지속가능한 학습 경험 설계

감성 분석 및 사용자 패턴 인식

기술: 자연어 처리(NLP), 시계열 분석

입력 데이터: 플랫폼 내 텍스트 입력, 이용 시간대, 학습 패턴

출력 결과: 사용자 감정 상태, 학습 몰입도, 최적 학습 시간

활용: 사용자 경험 최적화 및 정서적 지원 제공
